# Sistem E-Booking SK Bebuloh

A bilingual (English / Bahasa Melayu) room booking system for teachers of  
**Sekolah Kebangsaan Bebuloh, WP Labuan**.

## Key Features
- Teacher room booking (no login)
- Admin-only access via DELIMa Google account
- Prevents double booking
- Mobile-first design
- Glassmorphism UI (soft education theme)

## Tech Stack
- React (Vite)
- Supabase (Auth + Database)
- Vercel (Deployment)

> This project is built step-by-step with clarity and sustainability in mind.
